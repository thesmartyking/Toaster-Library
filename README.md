**Step 1. Add the JitPack repository to your build file ðŸ Š Add it in your root build.gradle at the end of repositories:**

To use this code enter this in your project to use in build gradle-**Project**
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

```

**Step 2. Add the dependency** ðŸ Š To use this code enter this in your project to use in build gradle-**Module**
```
dependencies {
	        implementation 'com.github.thesmartyking:Toaster-Library:1.1'
	}
```

**Step 3. Use Code ðŸ Š**
```
where context=this, Message="Your Message"
```

**1. Short Toast**

```@kotlin
ToastMsg().st(context,message)
```

**2. Long Toast**

```@kotlin
ToastMsg().lt(context,message)
```
